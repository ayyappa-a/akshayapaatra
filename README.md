# Akshaya Paatra

A premium single-page storefront concept inspired by the provided reference and brief.

## What is included

- Traditional Indian luxury visual system with deep green and terracotta sections
- Hero, promo banner, product grid, testimonials, supply map, and admin preview
- Search, category filters, product card interactions, and cart drawer behavior
- Static frontend only; backend integration points are left ready for Supabase, Razorpay, email, and SMS flows

## Run

Open `index.html` in a browser, or serve the folder with any static server.
